# 🎬 Movie Vault 

A dynamic, front-end web application that allows users to search for movies and save them to a personalized favorites list.

## 🚀 Live Demo
https://adityawinner7007.github.io/Movie-Vault/

## ✨ Features
- **React-Style Re-rendering:** Dynamically updates the UI without refreshing the browser using custom JS rendering logic.
- **Target Elimination:** Users can seamlessly remove movies from their list using the `filter()` method.
- **Data Persistence:** Integrated with Browser `localStorage` so your favorites are never lost, even after closing the tab.
- **Empty State Handling:** Clean UI feedback when no movies are in the list.

## 🛠️ Tech Stack
- **HTML & CSS:** For a clean, responsive layout.
- ** JavaScript :** Used for core logic, DOM manipulation, and state management.
